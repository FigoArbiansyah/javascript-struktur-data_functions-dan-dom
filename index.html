• Object

    • Pengenalan dan Cara Membuat Object

      • object dalam javascript ini sangat berguna dalam pengembangan aplikasi. Object juga bisa berisi beragam tipe data.

      • untuk membuat sebuah object pada sebuah variable, kita bisa menggunakan tanda kurung kurawal.

      • const user = {};

      • Object terdiri dari pasangan key dan value, key berperan mirip seperti variable dan value merupakan nilai yang bisa berisi tipe data apapun, termasuk object lain.

      • Info tambahan: Setiap key dan value bisa disebut property, dan setiap property dipisahkan oleh koma.

      • Cara mengakses nilai property pada object adalah, kita bisa memanggil nama object, kemudian diikuti tanda titik dan nama property nya.

        Contoh: user.firstname

      • Selain itu juga kita bisa memanggil property pada object dengan menggunakan tanda kurung siku

        Contoh: user['firstname']

      • Pemanggilan menggunakan kurung siku biasanya digunakan saat nama property mengandung spasi

        Contoh: user['alamat saya']

      • Meng-Assign ulang nilai property pada sebuah object

        Contoh: book.color = 'blue';

      • info tambahan: mengubah nilai berbeda dengan menginisialisasi ulang nilai

      • Menghapus sebuah property menggunakan delete
